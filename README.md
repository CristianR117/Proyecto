# Proyecto
Proyecto orientado a un sistema de cafetería de la Universidad Centroamericana José Simeón Cañas.

Consiste en un sistema interactivo, que contiene vista de productos, inicio de pedidos,
vista de los productos de un pedido, modificar el pedido y cancelar el pedido.

Tambien posee un historial del ultimo pedido realizado donde se ocupa el manejo de archivos
para guardalo como una factura simple y facil de entender.
